# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| 1.x.x   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities via GitHub Security Advisories.

## Security Measures

- Regular dependency audits
- Automated security scanning
- Code quality checks
- No hardcoded secrets
